# bgoc-transformer

## Quickstart

1. Export AWS credentials or edit cluster.json and playbook.json
2. `cd job && ./launch.sh`
3. Write down EMR Cluster ID
4. Edit project
5. Edit playbook to point Job to your data

Don't forget to shut-down the cluster!
